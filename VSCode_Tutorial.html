To install the VS Code setup, click on this link.


Then, you will need to click on the blue button for your operating system (Windows, Linux, or Mac).

Once you have clicked on the appropriate button, the setup guide for VS Code will start to download. Wait for that to happen, then follow the next steps.

1. Open the setup guide.

2. After reading the 'License Agreement' click the 'I accept the agreement' button and click next.

3. Next, you need to tick all of the boxes that you would like to use (I recommend using all in the 'other' section, as it means that almost all files in the file explorer can be launched in VS Code) then click next again.

4. Finally, click on the 'install' button, and VS Code will start to download as an usable application.
